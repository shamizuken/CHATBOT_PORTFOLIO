# 🔎 IA Portfolio Chatbot

Bienvenue sur le dépôt de mon assistant personnel intelligent ! Ce projet est un chatbot conçu pour aider les recruteurs à découvrir mon parcours, mes compétences et ma personnalité de manière interactive.



## 🚀 Le projet

L'objectif était de créer une interface conversationnelle capable de répondre aux questions sur mon profil professionnel tout en démontrant ma capacité à apprendre de nouvelles technologies rapidement.



## 🛠️ Stack technique

* **Langage :** Python (appris en autodidacte pour ce projet)
* **Interface :** Streamlit
* **IA :** API Groq (Modèle Llama 3.1)
* **Style :** CSS personnalisé et configuration TOML
* **Déploiement :** Streamlit Cloud



## 💡 Points Clés & Apprentissages

Ce projet démontre ma polyvalence au-delà de mon expertise initiale en **Développement Web Front-end (HTML/CSS/JS)** :
* **Apprentissage Rapide :** Transition du Web classique vers Python pour l'intégration d'IA.
* **Gestion d'API :** Connexion sécurisée à l'API Groq et gestion des requêtes.
* **Sécurité :** Mise en place de secrets d'environnement et d'un fichier `.gitignore` pour protéger les clés API.
* **UX/UI :** Personnalisation de l'interface Streamlit pour une expérience utilisateur soignée.



## 📁 Structure du dépôt
```bash

* `app.py` : Code principal de l'application.
* `requirements.txt` : Liste des dépendances Python.
* `.streamlit/` : Configuration du thème et gestion des secrets (local).

```
